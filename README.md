  <h1 style="color:#2c3e50;">📄 Merge Multiple HTML Files to Single PDF</h1>
  <p>
    This is a browser-based tool to upload multiple HTML files and merge them into a single downloadable PDF.
    Fast, free, and privacy-safe — runs fully on your browser.
  </p>

  <h2 style="color:#2c3e50;">🚀 Features</h2>
  <ul>
    <li>✔️ Upload up to <strong>45 HTML files</strong></li>
    <li>✔️ Supports both <strong>Portrait</strong> and <strong>Landscape</strong> modes</li>
    <li>✔️ Renders styles, images, and paginated content</li>
    <li>✔️ One-click PDF download</li>
    <li>✔️ 100% client-side — <strong>no server</strong> involved</li>
  </ul>

  <h2 style="color:#2c3e50;">📦 Tech Stack</h2>
  <ul>
    <li><strong>HTML5</strong>, <strong>CSS3</strong>, <strong>JavaScript</strong></li>
    <li><a href="https://github.com/parallax/jsPDF" target="_blank">jsPDF</a> for PDF creation</li>
    <li><a href="https://html2canvas.hertzen.com/" target="_blank">html2canvas</a> for DOM capture</li>
    <li><a href="https://pdf-lib.js.org/" target="_blank">PDF-lib</a> for PDF merging</li>
  </ul>

  <h2 style="color:#2c3e50;">📁 Usage</h2>
  <ol>
    <li>Open <code>index.html</code> in a browser</li>
    <li>Select up to 45 HTML files</li>
    <li>Choose orientation</li>
    <li>Click <strong>“Convert & Download Merged PDF”</strong></li>
  </ol>
